# ChronoPlayerController
The Player Controller used in the OFFICIAL Chrono Video Game

You may use this script but MUST credit me as Ben2790 and link to the github page

HOW TO USE

1. Add Scripts to your Unity 3d C# Project
2. Add a capsule and add 2 components to the base capsule (CharacterController & Rigidbody)
3. Add the player movement script to the capsule & name the capsule player
4. Put the Main Camera under the player capsule
5. Add the script to the Main Camera
6. Add a new tag called "Ground"
7. Make all objects that the player will come in contact with the newly created tag

And that is it.
